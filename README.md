# EnergyEye

## An IoT-Projekt

**Based on freeRTOS for esp:** https://github.com/SuperHouse/esp-open-rtos.git

**required Toolchain:** https://github.com/pfalcon/esp-open-sdk

### Build the Project

  export PATH=/path to esp-open-sdk/xtensa-lx106-elf/bin/:$PATH
  cd examples
  cd energyEye
  make
  make flash
